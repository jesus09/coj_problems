# coj_problems
Problems in COJ platform (Caribbean Online Judge)
==========

Programas realizados en la plataforma de juez en linea COJ.

http://coj.uci.cu/24h/problems.xhtml

Tipo de problemas
--------------------

Problemas de tipo como:

- Ad-Hoc
- Brute Force
- Data Structures
- Sorting-Searching
- Dynamic Programming
- Number Theory
- Strings
- Graph Theory

Lenguajes
--------------------

Los lenguajes encontrados pueden ser:
- Java
- C++
